## Download
 
 -  Add the following to your project level `build.gradle`:
 
 ```gradle
 allprojects {
	repositories {
		maven { url "https://jitpack.io" }
	}
}
```

 -  Add this to your app `build.gradle`:

```gradle
dependencies {
     compile 'com.github.sajad-zohrei:multi-directional-snackbar:1.1'
}
```
